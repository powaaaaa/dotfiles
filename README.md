# dotfiles

dotfiler を目指しています

# ディレクトリ構成

```
.
├── Makefile
├── README.md
├── .gitignore
├── home/
└── scripts/
```

- `Makefile`: `scripts/`配下の script を make コマンドで実行する
- `README.md`: これ
- [`home/`](./Home.md): `$HOME`配下と同じ構造になるように, dotfiles たちはこの中に配置
- [`scripts/`](./Script.md): 設定を簡単にする script を配置
